6. 加仓 / 减仓信号系统

加仓信号

  • AMD 放量突破前高或关键压力位。
  • 成交量大于 20 日均量 1.5x。
  • AMD 收盘站上 20 日线和 50 日线。
  • AMD 跑赢 SOXX / SMH / QQQ / NVDA。
  • AI GPU、Data Center、客户订单或分析师上调等基本面催化剂配合。
  • SPY、QQQ、SOXX/SMH 均处于风险偏好改善状态。

减仓信号

  • AMD 放量跌破 20 日线。
  • AMD 放量跌破 50 日线。
  • 高位放量滞涨或高开低走。
  • AMD 跑输 SOXX / SMH，说明个股相对弱。
  • 半导体板块放量下跌。
  • 大盘 risk-off，QQQ 和 SOXX/SMH 同时跌破关键均线。
