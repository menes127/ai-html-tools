利润率含义

第一季度非 GAAP 毛利率为 55%，第二季度指引约 56%。这支持 DCF 中一个关键假设：随着数据中心业务占比提升，AMD 的公司层面毛利率可以从 FY2025 的约 50% 向低至中 50% 区间迁移。

投资逻辑影响：正面 第一季度毛利率强劲 + 第二季度毛利率指引继续提升 = 数据中心 mix 和经营杠杆假设获得更多支持
需要注意：毛利率提升只是第一步。真正决定估值的是毛利率提升能否进一步转化为 EBIT margin 扩张，而不是被更高的研发投入完全抵消。

4. 为什么超预期？

收入超预期的原因

  1. 数据中心需求强于预期。
  2. EPYC 服务器 CPU 表现持续良好。
  3. Instinct GPU 出货继续放量。
  4. 客户端业务维持健康。
  5. 嵌入式业务开始企稳。

EPS 超预期的原因

收入超预期 + 产品组合改善 + 毛利率扩张 + 经营杠杆 = EPS 超预期

5. AI GPU 与数据中心解读

AMD 在本季度强调了 MI350、MI450 Series、Helios rack-scale AI systems 和 AMD Instinct GPU 出货的需求增强。

为什么重要：AMD 的牛市情景需要证明自己能成为 NVIDIA 之外的可持续第二 AI 平台。本季度还不能完全证明这一点，但确实强化了 AMD 在大规模 AI 基础设施中获得更高相关性的判断。
EPYC 服务器 CPU 份额提升 + Instinct GPU 放量 + MI450 / Helios 客户采用 + Hyperscaler AI 基础设施需求 = AMD 数据中心投资主线

更新后观点：本季度对 AMD 投资逻辑是正面的，但还不足以单独构成估值重估。AMD 仍需要连续多个季度证明数据中心增长和利润率扩张的持续性，才能缩小“当前盈利能力”和“估值隐含盈利能力”之间的差距。
